Why we need loops

These two code examples do nearly the same thing, one procedurally (line by line), and the other using a for loop. The for loop example has a built-in condition, and therefore can respond dynamically to whatever data you are dealing with. The first example prints undefined because it does not.

Procedurally

Without a for loop we add undefined variables. Also, much typing.

    For loop

    With a for loop we can add a condition. It is more efficient to type as well.
